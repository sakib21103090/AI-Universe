what id dom?
Ans:The Document Object Model is an application programming interface for HTML and XML documents. It defines the logical structure of documents and the way a document is accessed and manipulated.
What is the purpose of DOM in HTML?
Ans:The DOM defines a standard for accessing documents that allows programs and scripts to dynamically access and update the content, structure, and style of a document.

What are the different ways to select DOM elements in JavaScript?
Ans:
getElementById - search element by element_id.
getElementsByTagName - search element by tag name
getElementsByClassName - search element by class name.
getElementsByName - search element by name attribute.

What is event bubbling used for?
Ans:Event bubbling is a method of event propagation in the HTML DOM API when an event is in an element inside another element, and both elements have registered a handle to that event. It is a process that starts with the element that triggered the event and then bubbles up to the containing elements in the hierarchy

What is callback function in JavaScript?
Ans: callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action.
